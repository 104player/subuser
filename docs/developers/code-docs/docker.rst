Communicating with Docker
-------------------------

You should communicate with Docker via the DockerDaemon object.

.. automodule:: subuserlib.classes.dockerDaemon
 :members:
 :undoc-members:

The DockerDaemon object communicates with Docker both via Docker's HTTP API and by calling the Docker client directly.  The ``subuserlib.docker`` module is used when calling the Docker client directly.
